## Project Setup
Najprv nainštaluj všetky moduly príkazom
```sh
npm install
```

### Kompilácia a Hot-Reload development
Stránka sa ti bude automaticky refreshovať, ak spravíš zmeny v súboroch
```sh
npm run dev
```

### Kompilácia a minifikácia pre produkciu

```sh
npm run build
```
